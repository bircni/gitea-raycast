# Gitea Extension Help

## Setup

1. Open Raycast.
2. Run **Extensions**.
3. Select **Gitea**.
4. Open **Preferences** and set:
   - **Gitea Base URL** (required)
   - **Access Token** (optional, needed for private repos)

## Commands

- **Gitea Repositories**: Search repos and open sections.

## Tips

- Use **Refresh Repositories** if results look stale.
- If you see an error, verify the base URL includes `http://` or `https://`.

## Troubleshooting

- **No private repositories**: add a personal access token with correct scopes.
- **Invalid URL**: the base URL must include protocol.
